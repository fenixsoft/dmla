# 统计与概率论——AI转型数学准备的第三支柱

如果说[线性代数](../linear/introduction.md)是机器学习的"数据语言"——它告诉计算机如何表示和组织数据，[微积分](../calculus/introduction.md)是机器学习的"操作技巧"——它告诉计算机如何从数据中学习和改进，那么**统计与概率论**就是机器学习的"决策框架"——它告诉计算机如何在不确定的世界中做出理性的判断和预测。

本系列文章面向传统软件开发人员，系统介绍机器学习所需的概率统计知识，从随机变量、概率分布到统计推断、模型评估，再到 NumPy 实践和机器学习应用场景。
