## 快速开始

### 互联网访问

直接浏览本站，阅读公开文章。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/username/ideaspaces.git

# 安装依赖
npm install

# 启动开发服务器
npm run local
```

## 功能特性

- 📚 知识管理：Markdown 文章整理与发布
- 🎨 流程图支持：Mermaid 语法绘制流程图
- 💻 代码高亮：多语言语法高亮显示
- 🚀 本地沙箱：Python 代码在线运行
