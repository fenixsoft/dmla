# 神经网络结构

本章是深度学习系列的开篇，从神经网络的思想起源出发，逐步构建对神经网络结构的理解。我们将追溯从生物神经元到人工神经元的发展历程，理解感知机的学习能力与局限，掌握多层感知机的表达能力突破，并深入前向传播的计算机制。

## 目录

- [神经网络思想起源](./idea-origin.md) — 生物神经元启发、M-P模型、Hebb学习规则
- [感知机](./perceptron.md) — 线性可分性、学习算法、XOR局限
- [多层感知机](./mlp.md) — 隐藏层、万能逼近定理、表达能力
- [前向传播](./forward-propagation.md) — 信号流动、矩阵形式、计算图

## 学习目标

完成本章学习后，你将能够：

1. 理解神经网络思想的起源与历史脉络
2. 掌握感知机模型及其几何解释
3. 理解多层感知机如何突破感知机局限
4. 实现神经网络的前向传播计算

## 目标读者

- 技术水平：中级（有一定机器学习基础）
- 前置知识：线性代数、概率论、基础编程能力
